3 프로젝트

프로젝트는 개발자 이야기에서 가장 중요한 요소입니다. 개발자 이야기를 식당으로 비유한다면 프로젝트란 거기서 제공하는 요리와 같습니다. 이곳에서 당신이 수립하는 모든 계획이 진행되며, 아직 어린 학생이나 작은 스타트업을 위대한 개발자나 코스피에 상장한 위대한 기업으로 만들어내는 이야기가 만들어집니다.

해당 문서에서는 캐릭터나 프로젝트 타입에 관계없이 공통적인 이벤트, 특성, 보정치만이 소개됩니다. 각 캐릭터 및 프로젝트 특성에 따른 특성은 해당 페이지를 확인해주세요.

설명

정의 타입 설명 주요 이벤트
회사 (company) object 개발자를 고용하고 프로젝트를 생성
개발팀 (team) object 회사에 소속되고 작업을 수행하는 조직 프로젝트 관리
개발자 (developer) object 회사에 고용되고 개발팀에서 일함 능력치
프로젝트 (project) object 회사에 수익을 가져옴
작업 (task) object 다 완성하면 프로젝트가 완성됨 업무 난이도
관계도 (opinion) relation 개발자 간 관계 +/-
고용 (contract) relation 회사와 개발자의 관계 및 회사와 개발팀의 관계 임금
할당 (assign) relation 프로젝트와 팀의 관계
팀원 (member) relation 개발팀과 개발자의 관계
티켓 (ticket) relation 프로젝트와 작업의 관계
업무 (work) relation 작업과 개발자의 관계 작업 진행 상황
